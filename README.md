## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

## To Run json-server

In youy terminal:

```
npx json-server server.json -p 3333 -w

# or with delay of 2 seconds (test statics pages)
npx json-server server.json -p 3333 -w -d 2000
```
